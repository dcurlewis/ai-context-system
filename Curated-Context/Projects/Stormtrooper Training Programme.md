# Stormtrooper Training Programme

Initiative to improve Stormtrooper combat accuracy, which has been identified as a strategic liability. Leveraging new simulation technology from [[SienarTech]] to supplement live-fire exercises.

## Current Status
- **Phase**: Pilot programme running at Carida Academy
- **Metrics**: Targeting accuracy up from 12% to 18% in pilot cohort (target: 40%)
- **Vendor**: [[SienarTech]] holographic combat simulators deployed
- **Owner**: [[General Veers]] (executive sponsor)

## Key Milestones
- Pilot completion: end of Q1 2026
- Full rollout decision: pending pilot results
- Budget request for fleet-wide deployment: Q2 2026
